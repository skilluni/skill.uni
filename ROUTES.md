# Routes – SkillUni WebApp

## Public routes

- `/` – Homepage (hero, metrics, courses, about, testimonials, CTA).
- `/courses` – List of all courses.
- `/courses/:courseId` – Course detail with roadmap and lessons.

## Auth-related routes (future)

- `/login` – Login page (when auth is enabled).
- `/register` – Registration page (when auth is enabled).

## Utility / error routes

- `*` – Not Found page.
