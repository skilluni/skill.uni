# SkillUni WebApp

SkillUni is a non-profit technology education platform focused on free, board-aligned Java education for ICSE students in Classes 9 and 10.

This repository contains the WebApp that extends the existing YouTube presence (@skilluni) into a structured learning experience with courses, progress tracking, and an interactive roadmap.

## Goals (v1)
- Simple, trustworthy homepage that introduces SkillUni and its mission.
- Free enrollment into Class 9 and Class 10 Java courses.
- Interactive, playlist-backed learning paths.
- Metrics-based social proof using live YouTube data.

## Getting started
- Frontend: React + Vite.
- Backend/Auth/DB: Supabase (auth + Postgres).
- All services should run on free tiers or self-hostable options.

See `PRD.md` and `DESIGN.md` for detailed product and system design.
